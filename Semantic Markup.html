Semantic Markup

It is the markup that relates to the Meaning of content

Example of Semantic elelment: heading tags, paragraph, footer , header etc.

Example of None Semantic element : Div, Span etc.

Uses of semantic element

  1. Meaningful/ layout → structure
  2. Seo Freindly
  3. Readable + screen reader
